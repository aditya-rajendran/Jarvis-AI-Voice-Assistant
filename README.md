# Jarvis AI Voice Assistant

Jarvis is a Python-based AI voice assistant built using LiveKit for real-time audio communication.  
The project focuses on low-latency voice interaction, where spoken input is processed and answered by an AI system in real time.

This repository represents a practical exploration of real-time audio streaming, asynchronous programming, and AI-powered voice interfaces.

---

## Features

- Real-time voice input and output  
- AI-generated responses to spoken queries  
- LiveKit-based low-latency audio streaming  
- Modular structure for easy extension  
- Environment-based configuration for security  

---

## Tech Stack

- Python  
- LiveKit  
- AsyncIO  
- Large Language Model (LLM) backend (e.g. OpenAI)
